# embedded-tomcat
Example showing how to deploy a spring webapp into embedded tomcat as a single jar

## Dependency versions
Tomcat: 7.0.62
Spring Framework: 4.1.5.RELEASE

## How to launch

```
$ mvn package
$ sh target/bin/webapp
```
